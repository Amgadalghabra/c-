Project:
1. Calculation of a permissible game plan using backtracking.
2. Creating a database and table. Insertion and deletion of data. View the records in a table.
3. Map part with navigation and zoom buttons.
4. The shortest route for a game day

Description: Other useful hints

--- Requirements ---

> g++ v5.4 or newer
> QT Creator application

--- Compilation ---

The code can be compiled using the provided .pro and .ui files.

--- Execution ---

To start the program, we open all files (project) in the Qt Creator application 
and then we click "Run" or we press (Ctrl+R) on the keyboard.
